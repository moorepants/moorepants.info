There is another way to formulate this problem which is described in Ljung 1999 on page 561 for the same set of regressors for each predictor variable. If we define the \\( q \\) predictors, \\( y(t) \\), as linear functions of the unknowns, \\( \\theta \\), and the regressors, \\( \\varphi \\) then we get an equation that looks like this: .. raw:: html \[ y(t) = \mathbf{\theta}^T\varphi(t) \] where \\( y(t) \\) is a \\( q \\times 1 \\) vector, \\( \\mathbf{\\theta} \\) is a \\( p+1 \\times q \\) matrix of the unknown parameters, and \\( \\varphi \\) are the regressors which are a vector \\( p+1 \\times 1 \\).
